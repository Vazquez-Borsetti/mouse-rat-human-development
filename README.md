# Mouse–Rat–Human Development Dataset

Curated open dataset for translating developmental timing across mouse, rat, and human.

## Overview

This repository provides an open, curated dataset designed to facilitate cross-species translation of developmental timing between mouse, rat, and human. The project integrates heterogeneous and non-standardized data from multiple sources, enabling alignment of comparable biological events across species.

The dataset spans developmental stages from prenatal periods to adulthood and includes homologous developmental milestones such as sensory system maturation, neuroanatomical events, and behavioral changes, as well as processes from other physiological systems.

## Data Collection and Curation

Data were compiled and curated from previously published studies and heterogeneous sources. In addition, relevant information was systematically extracted from the scientific literature. A harmonization process was applied to standardize developmental events and enable cross-species comparisons.

## Use Cases

This dataset can be used for:

- Cross-species comparison of developmental timelines  
- Modeling developmental time translation  
- Evaluating scaling laws and deviations 
- Supporting research in developmental biology and neuroscience
